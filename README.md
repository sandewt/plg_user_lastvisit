# The login date plugin

*This plugin shows a message with the last login (visit) date after a login in the frontend.*

Note: more accurate would be the previous login date instead of the last login date.


# Configuration / Features

### INSTALLATION ###

- Download: https://github.com/sandewt/plg_user_lastvisit/releases

- Install and activate (or update) the plugin.

- Note: The Action Log - Joomla plugin should be enabled.


### FRONTEND MESSAGE ###

View of the last login date after a login.
The text can be changed with a language override, eg: "Last activity: ...."
